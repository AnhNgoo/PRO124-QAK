# Entities

This folder contains all game entities.

File hierarchy indicates things like inheritance, as in this example:

- item (folder)
	- item.gd
	- equipment (folder)
	- forageable (folder)
	- placeable (folder)
	- tools (folder)
		- tool.gd
		- tool.tscn
		- fishing (folder)
		- foraging (folder)
		- weapons (folder)
		- other (folder)
